### Requirements

- Docker
- Node >= 12

### Resources

- http://knexjs.org/
- https://vincit.github.io/objection.js/

1. In the root folder, run `docker-compose up`

```
postgres_1  | done
postgres_1  | server started
```

2. Run `yarn install` to install objection and knex.
